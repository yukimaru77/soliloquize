コマンド

sshコマンド

ssh接続を行う。ユーザー名はオプションで指定することもできる。

ssh [オプション] [ユーザー名@]接続先IPアドレスまたはDNSアドレス
オプション 説明
-l ユーザー名 指定したユーザー名でアクセスする。(ユーザー名@でも可)
-p ポート番号 ポート番号を指定する
-i ファイル名 秘密鍵を指定する

sshからログアウト

exitコマンド、logoutコマンド、Ctrl+Dでシェルが終了する。

Dockerコマンド

Dockerに関するコマンド

書式 説明
docker create Dockerイメージからコンテナを作成する
docker start コンテナを起動する
docker run createとstartを同時に行う
docker stop コンテナを停止させる
docker ps コンテナの一覧表示
docker images Dockerイメージの一覧を表示する
docker exec 指定したコンテナにコマンドを実行、シェルを開くときなど
docker pull docker hubからイメージをダウンロード。デフォルトだとlatest(最新版)のタグを使って最新版を取得

systemctlコマンド

systemdに関するコマンド

書式 説明
ststemctl start サービスを起動する
ststemctl stop サービスを停止する
ststemctl enable サービスの自動起動を有効にする
ststemctl disable サービスの自動起動を無効にする
ststemctl is-active サービスが起動しているか確認する
ststemctl status サービスの状態を確認する
ststemctl restart サービスを再起動する
ststemctl halt システムを停止しhalt状態にする
ststemctl reboot システムを再起動する
ststemctl poweoff システムを停止し電源を切断する

shutdownコマンド

システムの停止や再起動を行う

shutdown [オプション] 時間 [コメント]
オプション 説明
-h 省略した時のデフォルトのオプション。システムを停止する。
-r 再起動する。
-k シャットダウンは行わずにコメントのみを送る。
-c 予定されたシャットダウンを取り消す。
時間 +10で10分後(+は無くても可)、HH:MMでHH時MM分に実行、nowは今すぐに実行。

オプション 説明

x 制御端末のないデーモンなども表示
u ユーザー名も表示
f プロセスの親子関係をツリー状で表示
-e すべてのプロセスを表示。axオプションにあたる
-f 完全なフォーマットで表示
-p PID または p PID 指定したプロセスのみを表示
-l または l nice値も表示

killコマンド

プロセスにシグナルを送る。シグナルを送るとたいていのプロセスは終了するがそれはプロセスをどう実装したのかによる。このプロセスはHUPシグナルを受け取ったら終了すると実装されているから終了するのである。

kill [-シグナル名または-シグナル番号] プロセスID

または

kill -s [シグナル名またはシグナル番号] プロセスID
シグナル シグナルの内容
HUP(1) 端末が切断されたときのシグナル(プロセスを実行したユーザーがログアウトしたのと同じような意味合い)
INT(2) フォアグラウンドジョブがキーボードによる割り込み命令を受けたときのシグナル(Ctrl+C)
KILL(9) プロセスの強制停止のシグナル。クリーンアップせずに終了
TERM(15) クリーンアップして終了。指定しない場合はこれが送られる
CONT(18) 一時停止したプロセスを再開させるシグナル
STOP(19) プロセスを一時停止させるシグナル
TSTP(20) キーボードによる一時停止のシグナル(Ctrl+Z)

killall/pkillコマンド

特定のコマンドを実行しているすべてのプロセスにシグナルを送る。

killall(pkill) [-シグナル名または-シグナル番号] コマンド名

または

killall(pkill) -s(--signal) [シグナル名またはシグナル番号] コマンド名

catコマンド

ファイルの内容を標準出力に出力。複数の引数を指定可能。引数無しの場合は標準入力から入力を受け付ける。

cat [ファイル名1] [ファイル名2] …
オプション 説明
-n 各行の左端に行番号を付与

nlコマンド

指定した方法でヘッダ、本文、フッタに分けてそれぞれ行番号を付与。引数無しで標準入力からの受付になる。

nl [オプション] [ファイル名]

区切りとなる行は2文字で指定します。デフォルトは「\:」で、「\:\:\:」という行をヘッダの開始、「\:\:」という行を本文の開始、「\:」をフッタの開始行と見なします。区切り文字は「-d」オプションで変更できます。

オプション 説明
-b 形式 指定した形式で本文に行番号を付与
-h 形式 指定した形式でヘッダに行番号を付与
-f 形式 指定した形式でフッタに行番号を付与

オプションに指定する形式は以下のようなものがあります。

形式 説明
a すべての行
t 空白以外の行
n 行番号の付与中止
p パターン 正規表現にあう行のも行番号を付与

odコマンド

バイナリファイルの内容を8進数や16進数、ASCII文字で出力します。デフォルトでは8進数です。

od [オプション] [ファイル名]
オプション 説明
-t 出力タイプ 出力するフォーマットを指定する

出力タイプは以下のように指定します。

出力タイプ 説明
c ASCII文字で出力します。
o 8進数で出力します。
x 16進数で出力します。

headコマンド

ファイルの先頭部分を表示。デフォルトでは10行目まで表示します。

head [オプション] [ファイル名]
オプション 説明
-n 行数 先頭から指定された行数だけ表示
-行数 先頭から指定された行数だけ表示(非推奨)
-c バイト数 出力するバイト数を指定する

tailコマンド

ファイルの末尾部分を表示。デフォルトでは後ろから10行目まで表示します。

head [オプション] [ファイル名]
オプション 説明
-n 行数 末尾から指定された行数だけ表示
-行数 末尾から指定された行数だけ表示(非推奨)
-c バイト数 末尾から出力するバイト数を指定する
-f ファイルの末尾に追加された行だけを表示し続ける。(fg常駐、リアルタイムで更新)

cutコマンド

ファイルの各行から指定したフィールドを出力

cut [オプション] [ファイル名]
オプション 説明
-c 文字数 取りだす文字位置を指定する
-d 区切り文字 フィールドの区切り文字(デリミタという)を指定する
-f フィールド数 どのフィールドを知路出すか指定

pasteコマンド

ファイル1とファイル2の各行を連結して出力

paste [オプション] [ファイル名]
オプション 説明
-d 区切り文字 区切り文字を指定する。(デフォルトではタブ)

trコマンド

標準入力から読み込まれた文字列を変換したり、削除したりして標準出力に出力する。

tr [オプション] 文字列1 [文字列2]

なお文字列1と文字列2は同じ文字数でなければいけません。abcdeとghijkとした場合aはg、bはhというように1文字づつで対応しているからです。文字列1>文字列2なら多い分はすべて文字列2の最後の文字、文字列2<文字列1の場合は多い分は意味がなくなります。

オプション 説明
-d 文字列1でマッチした文字列を削除する
-s 連続する文字列1を一つにする(aaabbbのファイルを-s abならabになる。)

また文字列パターンにはクラスが使えます。文字列1にパターン、文字列2に何か文字指定か文字列1と2にlowerとupperの組み合わせ以外はエラーになりそうです(やってみたかんじなので正確ではありません、まあ文字列2にパターンの用途は大文字小文字変換以外にないと思いますが)

クラス 説明
[:alpha:] 英字
[:lower:] 英小文字
[:upper:] 英大文字
[:digit:] 数字
[:alnum:] 英数字
[:space:] スペース

sortコマンド

行単位でファイル内容をソートして標準出力に出力。デフォルトでは昇順

sort [オプション] [ファイル名]
オプション 説明
-b 先頭の空白は無視
-f 大文字小文字の区別を無視する
-r 降順にソートする
-n 数字を文字ではなく数値として処理する

splitコマンド

指定されたサイズでファイルを分割して新しいファイルとしてカレントディレクトリに出力。デフォルトでは100行で分割、フォルダ名の末尾にはaa、ab、acのように名前が付与されていく

split [オプション] [分割するファイル] [出力後のファイル名]
オプション 説明
-行数 指定された行数でファイルを分割

uniqコマンド

重複している行を調べて1行にまとめて出力する。入力するテキストストリームはソートして置く必要があるので多くの場合はsortコマンドとパイプでつなぎます。

split [オプション] [分割するファイル] [出力後のファイル名]
オプション 説明
-d 重複した行のみを出力
-u 重複していない行のみを出力

wcコマンド

ファイルの行数、単語数、文字数を表示する。デフォルトでは行数、単語数、文字数の順番で表示

wc [オプション] [ファイル名]
オプション 説明
-c 文字数(バイト数)を表示
-l 行数を表示
-w 単語数を表示
